# calculator

새 프로그래밍 언어를 배울 때마다 같은 사양의 사칙연산 CLI 계산기를 만들어 추가하는 저장소입니다.
같은 문제를 여러 언어로 풀어보면 문법이 아니라 **언어의 철학 차이**가 보입니다.

## 공통 사양

- `3 + 4` 형태로 입력 → `= 7` 출력 (반복 입력 루프)
- 지원 연산: `+` `-` `*` `/`
- `q` 입력 시 종료
- 오류 처리: 잘못된 입력 형식, 0으로 나누기, 지원하지 않는 연산자
- 숫자는 64비트 부동소수점(double / f64)으로 처리
